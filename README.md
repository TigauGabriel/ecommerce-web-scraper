A Python script that extracts product data (Title, Price, Stock) from e-commerce websites and exports it to CSV.

## Technologies Used
- Python 3.x
- BeautifulSoup4
- Requests

## How to use
1. Install dependencies: `pip install requests beautifulsoup4`
2. Run the script: `python scraper.py`
3. Check the output file `scraped_books.csv` for results.
